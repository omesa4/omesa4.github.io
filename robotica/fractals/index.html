Detector de Mentides

Mode 1: Sensor de pols

Pressiona el botó per fer una pregunta i detectar la veracitat mitjançant el sensor de ritme cardíac.

Mode 2: Aleatori

Pressiona el segon botó i el sistema determinarà aleatòriament si la resposta és certa o falsa.
